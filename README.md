## Installation
<a href="https://github.com/gormrgryn/todojh-todo-vue/releases/tag/0.0.1">Download app</a> <br/>
or <br/>
1. Clone the repository
```sh
   git clone https://github.com/gormrgryn/todojh-todo-vue.git
   ```
2. Install all the dependencies
```sh
   npm i
   ```
3. Run the project
 ```sh
   npm run dev
   ```
